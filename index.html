# Getting Started

The DMIT-NAIT organization has a Template that you can use to create the GitHub Web Pages site for your organization. It uses the VuePress-based static site generator.

  1. Click the green Use this Template button () above.
  2. Fill out the form for creating your new repository. Name it based on this pattern (replacing YourOrg with your organization's name) and using the same upper/lower case for your organization's name: YourOrg.github.io
  3. Follow the instructions in the TODO document for customizing your site.
